# Seven Photography 📸

منصة حجز خدمات التصوير الاحترافي — Professional Photography Booking Platform

## Features
- 📅 Interactive booking calendar
- 📦 Photography packages display
- 🖼️ Portfolio slideshow gallery
- 💬 WhatsApp direct contact
- 🔐 Admin dashboard with role-based permissions
- 🌙 Dark / Light theme
- 🌐 Arabic / English language support

## Tech Stack
- React 18
- Vite 5

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy
Connect this repository to [Vercel](https://vercel.com) or [Netlify](https://netlify.com) for instant deployment.
